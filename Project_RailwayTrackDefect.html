Railway Track Defect Analyzer

Overview:

I developed a website to analyze defects in railway tracks and predict maintenance needs. By aggregating data from visual inspections and ultrasonic testing, the website provides a comprehensive overview of track conditions.

Key Features:

  • Visual Representation of Defects
    • OBS (One cross with red paint)
    • IMR (Three crosses with red paint)
  • Detailed Actions for Defect Classification
    • Provides specific maintenance actions for each defect category
    • Assists track maintenance personnel in decision-making
  • Graphical and Tabular Data Analysis
    • Includes a pie chart showing types of probes used for testing
    • Displays data in both graphical and tabular formats

Technologies Used:

  • Backend: Django Framework
  • Frontend: HTML, CSS, JavaScript
  • Database: MySQL
