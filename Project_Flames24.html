Flames'24 - LBSCEK College Sports Website

Overview:

I created a dynamic website for our college's annual athlete meet, Flames 2024, hosted on Vercel. The site showcases all sports events, winners, and department-wise leaderboards. It also features a gallery with embedded Instagram posts and an admin dashboard for managing results.

Key Features:

  • Hosted on Vercel.
  • Showcases all sports events and results.
  • Displays department-wise leaderboards.
  • Features a gallery section with embedded Instagram posts.
  • Includes an admin dashboard to manage winners.

Technologies Used:

  • Backend: Django Framework
  • Frontend: HTML, CSS, JavaScript
  • Database: MySQL
