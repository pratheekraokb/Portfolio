ShopEase, an advanced eCommerce application

Overview:

ShopEase is an e-commerce website developed as a mini-project during my studies at LBS College of Engineering, Kasaragod. The project aimed to create a user-friendly and efficient platform for online shopping, integrating advanced features and technologies to enhance the user experience.

Key Features:

  • Login & Registration
  • Admin Panel
  • Home Page for Normal Users
  • Chatbot System
  • Product Display Page
  • MyCart & Checkout
  • Additional Features: Product brochure viewing, PDF download options, invoice bill viewing and PDF download, help section with tutorial videos, product recommendation system.

Technologies Used:

  • Backend: Django Framework
  • Frontend: HTML, CSS, JavaScript
  • Database: MySQL
  • Tools & APIs: Twilio Services API for notifications
